# laravel-blog

Blog application created with Laravel 5.5 and PostgreSQL


**Front-end features:**
* component based approach;
* BEM naming convention;
* Bootstrap 4;
* jQuery, ES6;
* SCSS;
* SVG sprites;
* Chart.js for blog stats;
